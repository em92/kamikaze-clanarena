# Kamikaze Clan Arena

### Description
Kamikaze Clan Arena is modification of Clan Arena gamemode which enables players to use kamikaze item.

### Supported maps:
- almostlost
- asylum
- bloodrun
- campgrounds
- eviscerated
- hearth
- hiddenfortress
- longestyard
- overkill
- quarantine

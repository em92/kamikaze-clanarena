# Kamikaze Clan Arena

### Description
Kamikaze Clan Arena is modification of Clan Arena gamemode which enables players to use kamikaze. Item spawns in 10 seconds after round stars.

### Supported maps:
- almostlost
- asylum
- bloodrun
- campgrounds
- eviscerated
- hearth
- hiddenfortress
- longestyard
- overkill
- quarantine

### Requirements
- minqlx v0.5.0 or above
